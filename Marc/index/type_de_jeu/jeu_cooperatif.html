Les jeux coopératifs


Les jeux coopératifs (ou de coopération) sont des jeux dans lesquels les joueurs jouent collectivement, sans compétition, afin de poursuivre un objectif commun. Soit l'ensemble des joueurs gagne s'ils atteignent leur but, soit ils sont tous perdants, solidairement, si ce n'est pas le cas.

Dans les jeux coopératifs, il faut faire attention et ne pas faire trop d'erreurs pour ne pas se faire taper par ses coéquipier. De plus il faut s'entourer de bons joueurs pour être sûr de gagner.

Si vous cherchez des jeux coopératifs, vous pouvez jouer à :

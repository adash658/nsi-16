The Game


The Game est un jeu de cartes coopératif de 1 à 5 joueurs dans lequel les joueurs s'unissent tous ensemble dans le but de parvenir à poser toutes les cartes sur 4 piles : 2 ascendantes, 2 descendantes. Pour les aider dans cette tâche, les joueurs pourront faire des sauts en arrière de 10. Mais cela suffira-t-il pour vaincre le jeu, sachant qu'ils ne pourront communiquer la valeur de leurs cartes en main ?

Le jeu s'appelle The Game parce que l'on joue ensemble contre le jeu. Les joueurs doivent coopérer, prendre des décisions ensemble et ne pas craquer quand la pression monte !

L'extension "On Fire" est incluse.
